# ES6 JavaScript Lab

A comprehensive collection of 7 JavaScript tasks demonstrating modern ES6+ features, built with a unified, professional **Glassmorphic** UI.

## 🚀 Overview

This repository contains solutions for the Level 3 ES6 JavaScript Lab. Each task is contained in its own directory (`t1` to `t7`) and features a beautifully styled, responsive glassmorphic dashboard interface. 

The lab focuses on practical applications of modern JavaScript concepts, shifting away from legacy syntax (like `var` or traditional function declarations) towards cleaner, more robust ES6 paradigms.

## 📁 Project Structure & Tasks

| Task | Directory | Topic / ES6 Features Covered | Description |
| :--- | :--- | :--- | :--- |
| **01** | `/t1` | `Class`, `Template Literals`, Array Methods | **Student Management System**: Displays a grid of student cards built dynamically using ES6 classes and template strings. |
| **02** | `/t2` | Destructuring, Spread Operator (`...`) | **Online Shopping Cart**: Extracts object properties via destructuring and calculates totals/max prices using spread syntax. |
| **03** | `/t3` | `Promises`, `async/await`, Fetch API | **Async Data Loader**: Fetches user data from a mock API, demonstrating modern asynchronous JavaScript handling. |
| **04** | `/t4` | `Set`, Spread Operator (`...`) | **Course Registration System**: Uses ES6 Sets to automatically filter out duplicate course entries. |
| **05** | `/t5` | `Map`, High-Order Functions | **Product Catalog**: Implements an ES6 Map for efficient key-value data storage, search, and deletion. |
| **06** | `/t6` | Class Inheritance, `Promise` | **Mini University Portal**: Simulates an async save operation and extends base classes for complex data handling. |
| **07** | `/t7` | `JSON.parse()`, `JSON.stringify()` | **Student JSON Manager**: Parses stringified JSON data into objects and displays both the UI visualization and raw JSON. |

## 🎨 Professional UI Design

All 7 tasks share a cohesive, premium design system:
- **Theme:** Deep navy background (`#0a0f1e`) with electric blue accents (`#60a5fa`).
- **Glassmorphism:** Elegant use of `backdrop-filter: blur()`, subtle semi-transparent borders, and soft inner shadows.
- **Layout:** Responsive grid system, sticky topbars, and clean data presentation tables.
- **Typography:** Built with the `Inter` font family for maximum legibility and a modern feel.

## 🛠️ How to Run

No build step is required! These are pure HTML/CSS/JS files.

1. Clone the repository.
2. Navigate to any task folder (e.g., `cd t1`).
3. Open `index.html` in any modern web browser.
   - *Tip: Use an extension like Live Server in VS Code for the best experience.*
