Tab 01

HTML — Structure

HTML (HyperText Markup Language) is the backbone of every web page. It defines the structure and meaning of content using a system of elements and tags.

  • Uses tags like <div>, <p>, <h1>, <a>
  • Semantic elements: <header>, <nav>, <section>, <footer>
  • Forms, tables, media embedding
  • HTML5 introduced canvas, video, audio
Tab 02

CSS — Style

CSS (Cascading Style Sheets) controls the visual presentation of HTML documents. It separates content from design, making websites beautiful and responsive.

  • Selectors: class, ID, pseudo-elements
  • Box model: margin, border, padding, content
  • Flexbox and CSS Grid for layouts
  • Animations and transitions
Tab 03

JavaScript — Behaviour

JavaScript adds interactivity to web pages. It runs in the browser and can manipulate HTML and CSS dynamically, handle events, and communicate with servers.

  • Variables, functions, loops, conditionals
  • DOM API for element selection and modification
  • ES6+: arrow functions, classes, promises
  • Async/Await for asynchronous operations
Tab 04

jQuery — Simplicity

jQuery is a fast, lightweight JavaScript library that simplifies HTML DOM traversal, event handling, animations, and Ajax with a simple API.

  • Cross-browser compatible selectors
  • Built-in animation methods: fadeIn, slideToggle
  • Method chaining for concise code
  • Plugin ecosystem for extended functionality
Tab 05

Ajax — Communication

Ajax (Asynchronous JavaScript and XML) enables web pages to fetch data from a server in the background without refreshing the page.

  • $.ajax(), $.get(), $.post() in jQuery
  • JSON is the modern data format of choice
  • RESTful APIs return data for consumption
  • Fetch API is the modern native alternative
